# terraform

## Install

```
tfenv install
```

## Usage

### Initialize

```
terraform init
```

### Plan

```
terraform plan
```

### Apply

```
terraform apply
```

## Environments

| Key | Value |
| ------------------ | ---------------- |
| TF_VAR_gcp_project | GCP's project ID |
| TF_VAR_gcp_region  | GCP's region     |
